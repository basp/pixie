# pixie
Basic .NET ray-tracer implementation as specified by 
the awesome [The Ray Tracer Challenge](https://pragprog.com/book/jbtracer/the-ray-tracer-challenge) book.

![reference image](https://i.imgur.com/LHdrSTH.png)
